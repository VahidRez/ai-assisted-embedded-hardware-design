# PCB Design Checklist

## Before layout
- Confirm all component values
- Recheck regulator datasheet
- Confirm package footprints
- Confirm connector orientation
- Confirm programming header pins

## During layout
- Keep power paths short
- Place decoupling capacitors close to IC pins
- Use a clear ground strategy
- Keep sensitive signals away from noisy paths
- Make debug access easy

## Before release
- Run ERC and DRC checks
- Review footprint sizes
- Review silkscreen labels
- Recheck mounting holes and connectors
- Export final fabrication files carefully
